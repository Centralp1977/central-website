# セクション別フォントサイズ・フォント名一覧の書き出し

指定されたページ（例: company.html）について、セクションごとのフォントサイズとフォント名を一覧にして書き出す。

## 手順

1. 対象ファイルの `<style>` 内を `grep -n "font-size"` と `grep -n "font-family"` で検索し、該当行をすべて洗い出す
2. `grep -n '<section\|id="'` などでセクション（`<section id="...">` や主要な `<div id="...">`）の境界を特定する
3. 各セクションのHTML本体を読み、どのCSSセレクタ（見出し・本文・ラベル・ボタン等）がそのセクション内の要素に対応するかを対応付ける
4. ヘッダー／モバイルメニュー／フッター／トップに戻るボタンなど、`<section>` に属さない共通パーツも別枠として洗い出す
5. `font-family` はページ全体で共通指定されていることが多いため、冒頭に一度だけ記載する
6. `clamp()` を使っている見出しは固定値ではない旨を明記する
7. 結果をMarkdownの表（セクション／要素／フォントサイズ）でユーザーに提示する。ファイルへの書き出しは指示があった場合のみ行う

## 注意点

- 依頼された対象ページのみを対象とする（他ページには手を広げない）
- モバイル用メディアクエリで値が変わる要素は「PC: Xpx／モバイル: Ypx」のように併記する
- 推測でサイズを埋めない。必ず実際のCSSをgrepして確認した値のみを記載する
